# 电商商城系统

基于 [芋道（ruoyi-vue-pro）](https://gitee.com/zhijiantianya/ruoyi-vue-pro) 二次开发的电商商城系统，在原有商城功能基础上新增**分销机制**与 **AI 成片**两大核心功能。

## 主要功能

### 商城核心
- 商品管理：SPU/SKU、分类、品牌、规格
- 订单管理：下单、支付、发货、售后、退款
- 营销活动：优惠券、秒杀、拼团、满减、积分
- 会员体系：等级、积分、签到

### 分销机制
- 多级分销：支持自定义分销层级，灵活配置佣金比例
- 分销员管理：申请、审核、团队查看
- 佣金结算：佣金记录、提现申请、结算审核
- 分销报表：业绩统计、团队数据分析

### AI 成片
- 商品视频自动生成：基于商品图片与描述，AI 自动生成营销短视频
- 多风格模板：支持多种视频风格与配乐选择
- 一键发布：生成后可直接关联商品，用于商品详情展示

## 技术栈

- **后端**：Spring Boot 2.7 + Spring Security + MyBatis Plus + Redis
- **前端**：Vue3 + Element Plus
- **移动端**：uni-app（支持小程序、H5、APP）
- **数据库**：MySQL
- **消息队列**：Redis / RabbitMQ
