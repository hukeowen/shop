推广机制

让顾客帮你带货，裂变增长

多级返佣 + 积分体系双管齐下，将老顾客变成推广大使，获客成本无限降低。

返佣推广

两级分销，自动结算

顾客 A 推荐 B 到店消费，A 获得一级佣金；A 的推荐人也能拿到二级佣金。全部自动计算，商户零工作量。

  • 商户自由设置一二级佣金比例
  • 各店推荐关系独立，互不干扰
  • 佣金冻结期可配置，防刷单
  • 用户随时申请提现，商户线上审核
积分体系

消费赚积分，积分当钱花

顾客每次消费自动累积积分，积分可在下单时直接抵扣金额，形成持续复购的正向循环。

  • 商户自定义积分兑换比例
  • 积分可转换为余额（不可逆）
  • 结算页显示可抵扣金额，提升转化
  • 积分流水全程可查，透明公正
